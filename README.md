# ARUINO - 반려동물과 보호자를 위한 스마트 펫 케어 시스템


# 프로젝트 소개
## 목표
- IoT 기술을 활용하여 반려동물의 건강과 생활을 관리할 수 있는 스마트 펫 관리 시스템 개발

## 주제 선정 배경
- 반려동물 시장의 성장
- 반려동물의 건강 문제 예방
- 보호자 돌봄 부담 완화
- 기술 융합의 가능성

## User Requirements

| **기능**                 | **설명**                                                                 |
|--------------------------|-----------------------------------------------------------------------|
| **배식 기능**            | - 사용자가 설정한 시간에 자동으로 배식<br>- 물 그릇에 자동 급수 및 단수                                   |
| **반려동물 상태 확인 기능** | - 착용한 목걸이를 통해 반려동물의 실시간 상태 확인                                           |
| **반려동물과 상호작용 기능** | - 반려동물의 모습을 웹캠으로 원격 관찰<br>- 보호자의 목소리를 출력해 반려동물과 소통<br>- 공 던지고 받는 놀이 기능 지원 |
